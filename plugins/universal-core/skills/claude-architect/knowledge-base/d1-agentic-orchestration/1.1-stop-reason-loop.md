# 1.1 · Agentic loop control via stop_reason
> **Domain:** D1 Agentic Architecture & Orchestration (27%)
> **Status:** PARTIAL · conf High · last-reviewed 2026-06-30
> **Source:** CCA-F audit 2026-06-29 (.claude/analysis/cca-f-capability-audit.html)

## What the exam expects
Drive the tool-use loop manually: branch on `stop_reason` (tool_use vs end_turn), append tool
results back to the message list, and re-call until the model stops — the primitive the harness
normally hides.

## What we have
Implemented as a spike: `~/.claude/spikes/agentic-loop-stop-reason/`. `engine.ts` hand-rolls the loop
(branches on every `stop_reason`; appends the whole assistant content; one user message of `tool_result`s
with matched ids); `fixloop.ts` adds the destructive-edit gate + differential-exit. Verified by **10
deterministic tests** (`node --test`) + `tsc --noEmit` clean — proves hand-rolling buys two interceptions
the SDK tool-runner can't. Findings: `.claude/analysis/cca-f-1.1-spike-findings.html`.
(Was: no in-repo loop — only one-shot `claude -p`, `skill-creator-patches/improve_description.py:174`.)

**Live run (2026-06-30):** added a `claude -p` adapter (`src/claude-cli-client.ts`) — shells the Claude Code
CLI (session auth, no API key) and translates its output into the engine's block protocol. `npm run smoke:cli`
drove the **same engine** end-to-end against a live model → `{status:"fixed", turns:3}` (read_error_context →
edit_file → run_test → fixed). Proves the loop + gate + differential-exit are real and provider-agnostic in
this key-less environment.

## Depth gap
The loop is now proven end-to-end against a live model — but via the `claude -p` shim, which **synthesizes**
`stop_reason`/`tool_use` from parsed JSON. The API's **native** `tool_use` blocks remain unexercised because
the raw Messages API is unreachable here: `igsenergy.com` is a managed Anthropic domain that blocks self-serve
org creation, and there is no API key — so both SDK auth paths are IT-gated. Clean BUILT needs that access,
then `npm run smoke` (SDK). → stays **PARTIAL**.
