Verdict — what an acquirer should note

1 · No dead code. All 62 skills have at least one incoming reference — there are zero true ghost skills. 5 sit at the floor of a single referencer (/manual-migration, /sync-vault, /prod-support-poller, /qe-dashboard, /regression-reporter); all are legitimate low-coupling roles (a migration helper, a deprecated redirect, a cron poller, a standalone dashboard, an end-of-run reporter), not orphans — though a single edge means one doc edit can silently strand them.

2 · Concentrated blast radius. Eight skills carry the ecosystem's coupling: /ready-for-testing (40 files), /three-amigos (34), /code-intelligence (28), /zephyr-reporter (25), /playwright-workflow (23). Renaming or changing the contract of any of these ripples across 20–40 files — these are the change-carefully nodes.

3 · Orchestrator/worker split is clean. 8 skills are pure sinks (referenced, reference nothing) — stable leaf workers/reporters. The conductors (/three-amigos out=13, /sync out=11, /ready-for-testing out=11) fan out to many. This is healthy layering, not tangle.

Method & Scope

how counted The reference contract

  • Unit: a slash-form mention /skill-name (≈95% of refs; also captures path form .claude/skills/skill-name/). Bare names without a slash are excluded (too noisy — sync, playwright, resume are common words).
  • Collision-safe: exact-name match with a trailing (?![\w-]) boundary. Proven necessary — naive /sync counts 105 mentions; the correct count is 43 (a 2.4× over-count avoided by excluding /sync-knowledge · /sync-repos · /sync-vault · /sql-schema-sync). /three-amigos 80 vs naive 88 (excludes /three-amigos-questions).
  • Incoming(X): references to X anywhere in scope except inside X's own folder (self-docs aren't coupling). Outgoing(X): references from inside X's folder to other skills.
  • Excluded (no silent caps): _shared/tickets/ (per request), _shared/_archive/ (deprecated defs), and the append-only skill-ledger*.md run logs (counted separately as run-counts below, not folded into reference totals). 253 files scanned (.md/.yml/.yaml/.txt/.ts.example).
  • Count discrepancy resolved: 62 skills, not 59 — Glob/find (case-sensitive) miss the 3 skills whose file is uppercase SKILL.md (comms-coach, playwright-cli, trace-call-path); they are confirmed registered skills.
